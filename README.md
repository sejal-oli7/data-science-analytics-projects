This repository contains Data Science and Analytics projects developed using Python and Jupyter Notebook.  
The projects focus on data cleaning, exploratory data analysis (EDA), data visualization, and statistical analysis to discover meaningful insights from real-world datasets.

Projects Included

1. Cafe Sales Analysis
- Data cleaning and preprocessing
- Handling missing values and inconsistent data
- Exploratory Data Analysis (EDA)
- Visualization of sales trends and patterns
- Extracting business insights from cafe sales data

### 2. King County House Sales Analysis
- Real estate dataset analysis
- Data preprocessing and feature exploration
- Statistical analysis
- Visualization of housing trends
- Understanding factors affecting house prices

3. Student Performance Analysis
- Student dataset exploration
- Data cleaning
- Performance pattern analysis
- Visualization of relationships between features
- Statistical insights from academic data

Skills Demonstrated

Purpose

The goal of these projects is to practice data analysis techniques and transform raw datasets into meaningful insights through visualization and statistical methods.
